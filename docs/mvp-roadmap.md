# Roadmap de implementação (MVP → V2)

## Fase 0 — Descoberta adaptada (3-5 dias)
- Mapeamento de fluxos padrão de micro negócio (sem entrevista inicial)
- Definição dos fluxos críticos para demo: venda, encomenda, caixa e resumo
- Protótipo de baixa fidelidade orientado a simplicidade
- Preparação de base de dados com dados fictícios realistas

## Fase 1 — Fundação técnica (1 semana)
- Setup de monorepo
- Setup de autenticação
- Modelo inicial da base de dados
- Layout base da aplicação

## Fase 2 — Operação principal (2 semanas)
- Módulo mini mercado
- Módulo encomendas de sopa
- Módulo finanças

## Fase 3 — Documentos e relatórios (1 semana)
- Recibos/faturas
- Resumo semanal e exportações

## Fase 4 — Qualidade e lançamento (1 semana)
- Testes de fluxo ponta a ponta
- Ajustes de UX/UI
- Backup, monitorização e formação rápida da utilizadora

## Melhorias para V2
- WhatsApp para confirmação de encomendas
- Previsão de stock com base no histórico
- Multi-loja/multi-operação
- Painel para contabilista
